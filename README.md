# LiteLoaderQQNT Plugin Template

## dependencies

如果需要使用该插件需要依赖 [scarletborder/LLQQNT-lib-moremenu](https://github.com/scarletborder/LLQQNT-lib-moremenu)

## feature

NTQQ设置中增加新配置项,可控制是否在关闭QQ主窗口后直接关闭QQ进程
